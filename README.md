# aura-luxe-jewelry-store 
A premium, responsive e-commerce frontend interface designed for a luxury handcrafted jewelry brand. This project focuses on delivering a sophisticated user experience through clean UI/UX, smooth micro-interactions, and functional e-commerce logic.

## 🚀 Live Demo
Check out the live website here: https://laiba-shaban.github.io/aura-luxe-jewelry-store/

🛠 Features
Modern Luxury Aesthetic: A dark-themed, gold-accented design tailored for high-end jewelry presentation.

Dynamic Shopping Bag:

Add products to the bag with instant count updates.

Real-time bill calculation using JavaScript.

Ability to remove items directly from the bag modal.

Interactive Checkout Flow: A user-friendly, multi-step form to capture customer details (Name, Address, Phone Number) before finalizing the order.

Category Filtering: Seamlessly filter products by category (Necklaces, Earrings, Rings, etc.) to improve discoverability.

Responsive Design: Optimized for different screen sizes to ensure a consistent experience.

💻 Technologies Used
HTML5: Semantic structure.

CSS3: Modern layout techniques (Flexbox/Grid), custom animations, and hover effects.

Vanilla JavaScript: DOM manipulation, state management for the cart, and form validation logic.
